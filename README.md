# Autonomous-Detection-of-Color-Vision-Deficiency
This project's goal was to design a gradient descent algorithm that can detect colorblindness and diagnose the distinct variety of it present by solving an optimization problem using approximations of the gradient, forward finite differences, penalty functions and a line search. These numerical techniques enabled us to minimize the steady-state visually evoked potential response and the amplitudes from the electroencephalogram's frequency spectrum. With this approach, I modified the stimulus vector to determine what color vectors the patient confuses and thus, diagnose their specific type of colorblindness. This project was completed with my partner, Jack Regovich, as part of a project for my MATH 317 Advanced Numerical Analysis course.
